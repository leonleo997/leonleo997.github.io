INFORMACIÓN

Componente Difultad
Tablas A la hora de pasar el mouse sobre una fila de la tabla se volvía blanco, y ya que el color de la letra era blanco este no se veía, para solucionarlo se modificó la clase hoover modificando el color al negro
Imagenes En la sección de intereses deseaba poner la imágen del gimnasio y el texto justo al lado de la imágen, con el objetivo de obtener el resultado esperado. para solucionarlo se usó Span ya que a diferencia de Div este no saltaba de línea
Google Fonts Se usó este conjunto de letras que sirvieron para agregar variedad. Esto debido a que el conjunto de letras que se encuntran por defecto no son muy variadas. Sin embargo a la hora de implementarla en ocaciones se debía usar !important debido a que por otras clases la letra estaba modificada por defecto, o en ocasiones, debido a la herencia, no se aplicaban los estilos.
Fontawesome Este se usó para adicionar íconos al navbar, se encontró dificultades como que los íconos quedaban debajo del texto que lo acompañaba
Modal Este componente se usó para crear ventanas emergentes, sin embargo, hubo problemas debido a que el botón de cerrar no se encontraba centrado. Para solucionarlo se creó una hoja de estilos donde se centraba el boton del Modal
Nabvar Para esta herramienta no hubo problemas debido a que se manejó en clase.
Scrollspy En este componente se presentó la mayor cantidad de problemas, pues para iniciar se debió ajustar el tamaño de las imágenes a un ancho y alto fijo. También, al tratar de centrar el componente se presentó problemas.
ButtonGroups Para este componente se presentó el mismo problema que en las tablas y se solucionó de la misma manera. También, encontramos problemas debido a que este contiene conjuntos de botones y se reemplazó por enlaces
Panels Este componente fue usado en vez de un Div con la clase Contenedor. Modificando el nivel de opacidad más claro para un mejor efecto
Jumbotron Con este componente no se presento problemas pues ya que se trabajó en clase
